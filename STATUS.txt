🔐 SECURITY SUMMARY

  Hardware ID Generation:
  └─ Input: MAC(s) + hostname + CPU model + platform
  └─ Algorithm: SHA256
  └─ Output: 16 uppercase hex characters
  └─ Example: A1F4B9E2C7D3F501

  License Key Generation:
  └─ Input: Hardware ID
  └─ Algorithm: HMAC-SHA256(Hardware ID, Secret)
  └─ Output: 20 characters, formatted XXXX-XXXX-XXXX-XXXX-XXXX
  └─ Example: K9X2-F7M4-J3Q8-P1L6-W9Z5

  Data Protection:
  └─ Database: Atomic writes, no partial saves
  └─ Sync: HTTPS encryption in cloud
  └─ API: IPC isolation, contextBridge security
  └─ License: Device-specific anti-piracy
